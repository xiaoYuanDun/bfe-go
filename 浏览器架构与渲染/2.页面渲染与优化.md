- 渲染页面的主要步骤

  sytle --> js --> layout --> (layer) --> paint --> composite

  cssom --> dom --> 布局 --> 分层 --> 绘制 --> 合成

- 优化中重要的三个点

  重排, 重绘, 合成

- 为何动画比 js 高效

  js 操作基本上一定会触发重排, 而重排一定会触发重绘, 重绘一定会触发合成, 重排的性能最差的

  动画会创建一个新的图层, 他的更新不会触发 重排, 重绘, 直接在合成线程上进行合成操作, 不会对主线程造成任何影响, 是性能最好的
